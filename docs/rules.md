# 鱼盆模型操作规则

## 选股规则

### 板块筛选标准
1. 成交额排名前20的板块
2. 板块内涨停股数量 >= 3
3. 板块指数处于上升趋势
4. 资金净流入为正

### 个股筛选标准
1. 属于强势板块
2. 市值适中（50亿-500亿）
3. 换手率在合理区间（3%-15%）
4. 技术形态良好

## 买入信号

### 板块信号
- 板块整体走强
- 龙头股涨停或接近涨停
- 板块内多只个股联动上涨

### 个股信号
- 突破关键压力位
- 量价配合良好
- 均线多头排列

## 卖出信号

### 止盈规则
- 短线: 10%-20%收益
- 中线: 30%-50%收益
- 根据板块热度调整

### 止损规则
- 固定止损: -7%
- 板块退潮: 及时止损
- 破位止损: 跌破关键支撑

## 仓位管理

1. 单只个股不超过总仓位的20%
2. 单一板块不超过总仓位的40%
3. 保留20%现金应对机会

## 风险控制

- 设置严格的止损线
- 避免追高
- 关注大盘风险
- 控制持仓数量（5-10只）
