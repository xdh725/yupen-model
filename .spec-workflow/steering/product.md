# Product Overview

## Product Purpose

鱼盆模型 (Yupen Model) 是一个基于技术分析的股票市场趋势判断工具。该产品通过计算指数当前价格与20日均线的偏离度，判断市场处于牛市还是熊市信号，帮助投资者把握板块轮动节奏和市场趋势。

核心理念是将股市比喻为"鱼盆"，资金在不同板块间流动如同鱼在盆中游动，通过识别资金流向来捕捉投资机会。

## Target Users

### 主要用户
- **A股个人投资者**: 需要简单直观的市场趋势判断工具
- **技术分析爱好者**: 关注均线系统和趋势跟踪策略
- **量化交易初学者**: 学习和实践基础的趋势跟踪模型

### 用户痛点
- 市场信息过载，难以快速判断整体趋势
- 缺乏简单有效的买卖信号参考
- 需要同时跟踪多个指数的走势
- 希望验证交易策略的历史表现

## Key Features

1. **指数状态监控**: 实时显示多个指数的牛市/熊市信号状态
2. **强度排名**: 按偏离度对指数进行强度排名，识别最强/最弱指数
3. **市场态势分析**: 统计整体市场的牛熊比例，判断市场氛围
4. **历史回测**: 支持导入历史数据，验证鱼盆模型的信号表现
5. **数据导入**: 支持通过 CSV 文件导入指数数据

## Business Objectives

- 提供简单易用的市场趋势判断工具
- 帮助用户快速识别市场整体方向
- 降低投资决策的信息获取成本
- 提供可回测的策略验证功能

## Success Metrics

- **信号有效性**: 模型发出的买卖信号历史准确率
- **用户活跃度**: 用户访问和使用频率
- **功能使用率**: 回测功能的使用次数和完成率
- **数据覆盖**: 支持跟踪的指数数量

## Product Principles

1. **简洁直观**: 界面清晰，信号明确（YES/NO），无需复杂解读
2. **数据驱动**: 所有判断基于客观数据计算，避免主观猜测
3. **可验证性**: 提供回测功能，用户可验证策略历史表现
4. **风险提示**: 明确标注模型仅供参考，不构成投资建议

## Monitoring & Visibility

- **Dashboard Type**: Web-based 单页应用 (SPA)
- **Real-time Updates**: 手动刷新/CSV导入更新
- **Key Metrics Displayed**:
  - 各指数当前价格 vs 临界值
  - 偏离度百分比
  - 牛市信号数量/占比
  - 最强指数识别
- **Sharing Capabilities**: 支持导出 CSV 数据

## Future Vision

### Potential Enhancements

- **实时数据源**: 接入第三方 API 实现自动数据更新
- **更多指数**: 支持行业指数、海外指数等
- **历史图表**: 可视化历史价格和信号变化
- **策略优化**: 支持自定义参数（均线周期、浮动比例等）
- **预警功能**: 信号变化时推送通知
- **移动端**: 开发移动应用或响应式设计
